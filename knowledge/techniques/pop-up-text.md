# Pop-Up Text / Letters

**Difficulty:** Beginner–Intermediate
**Best for:** "Happy Birthday", "I Love You", names, single words as the card's focal point

## How It Works

Each letter is a V-fold element designed as a symmetric shape foldable in half vertically. Letters are individual pieces, each glued as a V-fold along the gutter, spaced to spell the word. The word rises up together when the card opens.

## Geometric Constraints

- Letter height: 20–50mm (maximum = half the card width)
- Minimum stroke width in any letter: **5mm** (thinner strokes tear)
- Use block letters — avoid thin serifs or fine details
- Spacing between letters: 5–10mm
- Total word width must not exceed card length − 20mm
- Each letter is its own V-fold piece — they are not connected to each other

## Letter Design Rules

- Each letter must be **symmetric** when folded in half vertically — the fold line runs down the center of the letter
- Letters with natural symmetry (A, H, I, M, O, T, U, V, W, X, Y) work best
- Asymmetric letters (B, C, D, E, F, G, etc.) need to be adapted: design them as block shapes that read correctly from the front when standing
- Avoid letters that are too thin when folded (e.g., thin capital I — give it a serif base for stability)

## Step-by-Step

1. Plan the word on paper — calculate total width and confirm it fits the card
2. Design each letter as a flat shape; mark the center fold line
3. Cut all letters from 80 lb cardstock (same color or varied for visual interest)
4. Score and fold each letter in half at its center fold line
5. Position letters along the gutter, spacing evenly
6. Glue one half of the first letter to the left side of the base, aligned with the gutter
7. Close card to bond the other half; open and check alignment
8. Repeat for each subsequent letter, working outward from center

## Template Markup

- Provide a flat template for each unique letter shape
- Mark center fold line on each letter
- Mark glue tabs (hatched) at base of each letter half
- Show spacing guide on base card (positions for each letter along gutter)

## Common Uses in Scenes

- Full word as centerpiece of an otherwise simple card
- Single initial combined with a figurative scene
- Message text above a box-fold platform
