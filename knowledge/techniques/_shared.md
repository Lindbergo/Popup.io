# Shared Reference

## Glossary

| Term | Definition |
|---|---|
| **Gutter** | The center fold line of the base card — the spine |
| **Valley fold** | Paper dips down (like a valley). Shown as dashed lines on templates |
| **Mountain fold** | Paper peaks up (like a mountain). Shown as dot-dash lines on templates |
| **Score line** | A pressed crease made before folding, required for clean folds on cardstock ≥ 80 lb |
| **Pedestal** | A strip of paper glued perpendicular to a surface that holds a floating element up |
| **Tab** | A flap of paper used for gluing one piece to another |
| **Pop-up element** | Any piece of paper that rises up when the card is opened |

## Paper Specifications

### Recommended Paper Weights

| Use | Weight | Notes |
|---|---|---|
| Base card | 110 lb (300 gsm) | Sturdy, holds mechanisms firmly |
| Pop-up elements | 80–90 lb (216–245 gsm) | Flexible enough to fold, strong enough to hold shape |
| Small detail pieces | 65–80 lb (176–216 gsm) | For decorative elements that don't bear load |
| Photo inserts | Any weight | Glued flat onto pop-up elements — no structural role |

### Standard Card Sizes

| Format | Flat dimensions | Folded dimensions |
|---|---|---|
| A2 (US standard) | 5.5" × 8.5" | 4.25" × 5.5" |
| A6 | 4.5" × 6.25" | 3.125" × 4.5" |
| Square (small) | 6" × 6" | 3" × 6" |
| Square (large) | 8" × 8" | 4" × 8" |

### Tools Required (all designs)

- Scoring tool or bone folder
- Ruler (metal edge preferred)
- Craft knife or X-acto knife
- Cutting mat
- PVA glue or double-sided tape
- Pencil

## Universal Design Rules

These apply to ALL techniques. A design that violates any of these will not function.

1. **Nothing extends beyond card edges when closed.** Every pop-up element must fold flat within the card footprint.
2. **All score/fold lines must be straight.** Curved shapes are achieved by cutting, not folding.
3. **Cuts run from valley to mountain to valley** — they always cross a fold line.
4. **Fold lines alternate** between mountain and valley — never two mountains or two valleys adjacent.
5. **Test before decorating.** Build the mechanism, test it opens and closes 3+ times, then decorate.
6. **Glue placement is critical.** Too close or too far from the gutter and the mechanism binds or collapses.

## Template Color Convention

| Color | Line type | Meaning |
|---|---|---|
| Red | Solid bold | Cut line — cut all the way through |
| Blue | Dashed | Valley fold — fold toward you |
| Green | Dash-dot | Mountain fold — fold away from you |
| Gray | Solid thin | Score line (press, don't cut) |
| Yellow fill | Hatched area | Glue zone — apply adhesive here |

## Difficulty Levels

| Level | Mechanisms | Estimated time | Tools |
|---|---|---|---|
| Beginner | Single V-Fold or Box Fold only | 20–40 min | Basic (knife, ruler, glue) |
| Intermediate | 2–4 mechanisms combined | 45–90 min | Basic + scoring tool |
| Advanced | 5+ mechanisms, moving parts | 90–180 min | Full tool set + brad fasteners |

## Failure Modes & Fixes

| Problem | Likely cause | Fix |
|---|---|---|
| Element doesn't pop up fully | Glue tabs too short / too far from gutter | Move glue attachment closer to gutter |
| Card won't close flat | Element too tall or extends past card edge | Reduce height or trim element |
| Fold is lumpy | Cardstock not scored before folding | Always score first, then fold |
| Element tears at fold | Paper too thin for mechanism size | Use heavier paper or reduce element size |
| Box fold collapses | Connecting fold not reverse-folded firmly | Crease harder; ensure opposite direction to main fold |
| Pedestals tilt layer | Pedestals not identical height | Re-cut all three to exactly equal height |
| Pull-tab jams | Slit too narrow | Widen slit by 0.5mm on each side |

---

## Multi-Mechanism Assembly Order

When combining mechanisms, always build in this order:

1. Background (floating layers, back panels)
2. Deepest / tallest structures (box folds furthest from viewer)
3. Mid-ground elements (v-folds, angle folds)
4. Foreground elements (chairs, figures)
5. Surface decorations and photo inserts last
