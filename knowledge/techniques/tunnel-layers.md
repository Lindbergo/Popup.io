# Tunnel / Layered Depth

**Difficulty:** Intermediate–Advanced
**Best for:** Forest scenes, hallways, theater stages, any scene requiring strong depth perspective

## How It Works

Multiple flat panels are stacked at increasing distances from the gutter, each with a cutout window. Looking through the stacked cutouts creates a tunnel perspective. Each layer is attached to the base card at its bottom edge with a fold tab. Layers fold down flat independently when the card closes.

## Geometric Constraints

- Layer 1 (front): closest to viewer, largest cutout, attached closest to gutter
- Each subsequent layer: progressively further, smaller cutout, attached further from gutter
- Spacing between layers: 10–20mm (enough to see each layer clearly)
- Each layer's cutout must be smaller than the cutout in the layer in front of it
- Total stack depth must fit within card height when closed — all layers fold flat
- Each layer folds down independently; design fold tabs so layers don't obstruct each other

## Step-by-Step

1. Design layers from back to front: start with the background (solid), work forward adding progressively larger cutouts
2. Cut all layer panels to the same width as the card interior; vary height as needed
3. Score and fold the attachment tab at the bottom of each layer
4. Attach the back layer first (furthest from gutter), gluing its tab to the base card
5. Work forward, attaching each layer at its correct distance from the gutter
6. Test that each layer folds flat cleanly before attaching the next
7. Add decorative elements to each layer surface (trees, buildings, figures, etc.)

## Template Markup

- Show all layers in an exploded side-view diagram
- Label each layer (Layer 1 Front, Layer 2, etc.) and its distance from gutter
- Mark cutout outlines on each layer
- Mark fold tab at base of each layer

## Common Uses in Scenes

- Forest: each layer has progressively smaller tree silhouettes
- Theater: proscenium arch front layer, mid-stage, backdrop
- Hallway: door frames getting smaller toward vanishing point
- Underwater: coral layers getting sparser toward the back
