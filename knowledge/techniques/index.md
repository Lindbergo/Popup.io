# Techniques Index

Quick reference: which mechanism to use for which scene element.

| Technique | File | Best for | Difficulty |
|---|---|---|---|
| V-Fold | v-fold.md | Trees, flames, hearts, letters, simple rising shapes | Beginner |
| Box Fold | box-fold.md | Tables, platforms, stages, buildings, shelves | Beginner–Intermediate |
| Floating Layer | floating-layer.md | Background panels, sky layers, depth illusions, image frames | Intermediate |
| Angle V-Fold | angle-v-fold.md | Leaning/pivoting elements, characters turning to face viewer | Intermediate |
| Moving Arm | moving-arm.md | Waving hands, opening doors, clock hands, nodding heads | Intermediate–Advanced |
| Sliding Motion | sliding-motion.md | Characters entering from side, sliding doors, text reveals | Intermediate |
| Rotating Disc | rotating-disc.md | Clock faces, wheels, spin-the-wheel interactives | Advanced |
| Tunnel Layers | tunnel-layers.md | Forest scenes, hallways, theater stages, depth illusions | Intermediate–Advanced |
| Pop-Up Text | pop-up-text.md | "Happy Birthday", names, single words as focal point | Beginner–Intermediate |

## Scene → Mechanism Quick Map

| Scene element | Use this |
|---|---|
| Table, platform, stage | box-fold |
| Chair (any angle) | v-fold |
| Person sitting on chair | v-fold element on top of chair v-fold |
| Person standing | v-fold |
| Tree / plant | v-fold |
| Building | box-fold (stacked) |
| Sky / background panel | floating-layer |
| Text message | pop-up-text or floating-layer |
| Door that opens | moving-arm |
| Character entering from side | sliding-motion |
| Wheel / clock | rotating-disc |
| Forest / hallway scene | tunnel-layers |
| Leaning or angled element | angle-v-fold |
